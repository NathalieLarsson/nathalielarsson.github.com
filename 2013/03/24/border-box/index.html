Border-box all the things

and it can save your ***

Posted by Nathalie Festin on March 24, 2013

One of the most annoying things with css is the relationship between width and padding.

This code would result in an 120px x 120px div:

div{
	width: 100px;
	height: 100px;
	padding: 10px;	
}

This code would result in an 100px x 100px div:

div{
	box-sizing: border-box;
	width: 100px;
	height: 100px;
	padding: 10px;	
}
* { 
	-moz-box-sizing: border-box; 
	-webkit-box-sizing: border-box; 
	box-sizing: border-box;
}
